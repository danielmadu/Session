## Session 1.0

Classe para manipulação de sessão no PHP


## Recursos
  - get($key)
  - set($key, $value)  
  - debug()
  - clear()
  - destroy()
  - remove($key)
  - isRegistered()
  - getSessionId()



## Utilização via composer

```sh
    "repositories": [
        {
            "type": "git", 
            "url" : "https://github.com/tayron/Session.git",
            "reference": "origin/master"
        }
    ],
    "require": {
        "tayron/session" : "master"
    },    
```
